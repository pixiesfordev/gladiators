This method will be called after all common fields from within the InspectorFields() callback have been displayed. You should implement your own inspector fields here.

Due to how Unity's undo system works you should only display the fields here and store the results in a private variable. Then assign the results inside the AssignSpecific method.

// Specific fields for the parallelepiped renderer
private Vector3 from, to;
protected override void SpecificFields() {
from = EditorGUILayout.Vector3Field("From", _renderer.From);
to = EditorGUILayout.Vector3Field("To" , _renderer.To );
}
