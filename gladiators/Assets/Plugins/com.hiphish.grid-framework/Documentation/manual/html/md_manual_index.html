Thank you for choosing Grid Framework for Unity3D. This document is the official user manual and scripting reference. If you are new to Grid Framework you should first read the user manual to get familiar with the concepts of grids and how they work in Grid Framework. From there on you can examine the included examples or jump straight into the scripting reference part of the manual.

Additional extensions

Grid Framework is a core library upon which more higher-level tools can be built. The following official extensions are available as Unity3D Git packages:

  • Playable samples: Short playable examples to get started
  • Grid align panel: an editor extension for aligning and scaling objects in a scene
  • Playmaker support: Playmaker bindings for Grid Framework let you use grids in Playmaker's visual scripting
  • Vectrosity backend: a rendering backend using the popular Vectrosity library

Please refer to each of the packages for more information and installation instructions.
