Detailed Description

This struct is based on Unity's own Vector3 struct and can be used in a similar way. It resides in Grid Framework's own namespace to prevent collision with similar types from other plugins or a future official Unity Vector6 type.

The API is mostly the same as for Vector3, except in places where it wouldn't make sense. The individual components are named to reflect their shearing nature according to the following pattern: ab where a is either X, Y or Z and b is another axis different from a. Two adjacent values belonging to the same axis can also be accessed as a Vector2, like for example YX and YZ, but not XZ and YX.

The individual components are indexed as follows: XY=0, XZ=1, YX=2, YZ=3, ZX=4, ZY=5.
