Detailed Description

In order to be visible in the scene a grid has to be rendered by a renderer. A grid renderer is similar to Unity's own MeshRenderer class in that it does only the displaying job.

The shape of the rendered grid depends on the renderer used, to change the shape you have to assign a different renderer to the GameObject. This class is the base class for all renderers, similar to how RectGrid is the base class of all grids.
